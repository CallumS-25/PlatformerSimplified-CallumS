How to access the project:
1.	Click the “Code” button.
2.	Download the ZIP file and extract the files anywhere accessible on your PC. (preferably Desktop)
3.	Make sure you have Epic Games Launcher AND Unreal Engine 5.5.4 installed.
4.	To access the editor code of this project, simply go into the “PlatformerSimplified-CallumS” file and click on “PlatformerSimplified.uproject” file.
(Most of the coding can be found in the “Platformer” file in the project.)

PLEASE NOTE: STATE TREES IN THIS VERSION OF UNREAL ENGINE ARE REALLY BUGGY AND CAN POTENTIALLY CAUSE CRASHES ON STARTUP. 
SIMPLY RESTART LOADING THE PROJECT, AND IT SHOULD EVENTUALLY LOAD.
